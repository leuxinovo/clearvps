## 前言
komari 在随着使用的时间 数据库占用硬盘的大小也在随之增加 利用sqlite3 来清理数据库大小
## 安装sqlite3
```bash
    apt install sqlite3
```
## 停止komari
```bash
    systemctl stop komari.service
```
## 备份komari
防止修改数据库出现问题 修改前请备份 防止数据丢失
```bash
    cp /opt/komari/data/komari.db /opt/komari/data/komaribf.db
```
## 打开数据库
```bash
    sqlite3 /opt/komari/data/komari.db
```
### 删除记录
```bash
    DELETE FROM records;
```
### 删除长期保存的记录
```bash
    DELETE FROM records_long_term;
```
### 删除延迟记录
```bash
    DELETE FROM ping_records;
```
### 收缩数据库大小
```bash
    VACUUM;
```
### 退出数据库
```bash
    .exit
```
## 重启komari
```bash
    systemctl restart komari.service
```
